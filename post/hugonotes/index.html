紀錄如何使用Hugo

建立及設定部落格專案 我們先使用 hugo 命令新增一個空白專案，然後下載一個Template到我們的專案裡面， 接著新增四個我們想加到模板 Menu 的頁面: about, history, tags, categories. 最後則是新增一篇空白的文章到專案內。

1.create the project

$ hugo new site myblog

2.add a theme
$ git submodule add https://github.com/laozhu/hugo-nuo themes/hugo-nuo

3.add new pages

$ hugo new about.md
$ hugo new hisroty.md
$ hugo new tags.md
$ hugo new categories.md

4.add new article

$ hugo new post/welcome.md
